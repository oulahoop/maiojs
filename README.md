# MaioJS

## How to play 

Go to [MaioJS](https://oulahoop.github.io/maiojs/)
- Use the WASD or ZQSD keys to move the player
- Use the space key to jump

## How to install locally

- Install the dependencies

```bash
cd maiojs
npm install
```

- Start the development server

```bash
npm run dev
```
